# Collectors Module (Problems 95–106)

## Mnemonic
**“Stats Partition Group Avg Count Oldest Gender Names Students Partition City.”**

- Stats → Summary statistics (95)
- Partition → Even/odd partition (96)
- Group → Employees highest paid (97)
- Avg → Employees average salary (98)
- Count → Employees count (99)
- Oldest → Employees oldest/youngest (100)
- Gender → Employees avg age by gender (101)
- Names → Employees names by dept (102)
- Students → Students avg grade (103)
- Count → Students count (104)
- Partition → Students partition by grade (105)
- City → Students grouped by city, filter Hyderabad (106)

## Key Focus
- Collectors for summarizing, partitioning, grouping
- Advanced grouping with mapping, maxBy, averaging
- Practical scenarios: employees and students datasets
