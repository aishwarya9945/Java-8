# Strings Module (Problems 26–45)

## Mnemonic
**“Reverse Pal Vowel Consonant Dup Space Word Long Short Freq Ana Unique First Char Rot Rev Cap Sub Rep Char.”**

- Reverse → Reverse string (26)
- Pal → Palindrome (27)
- Vowel → Count vowels (28)
- Consonant → Count consonants (29)
- Dup → Duplicate characters (30)
- Space → Remove whitespace (31)
- Word → Count words (32)
- Long → Longest word (33)
- Short → Shortest word (34)
- Freq → Word frequency (35)
- Ana → Anagram check (36)
- Unique → Remove duplicates (37)
- First → First non-repeated char (38)
- Char → Character frequency (39)
- Rot → Rotation check (40)
- Rev → Reverse words (41)
- Cap → Capitalize words (42)
- Sub → Substring check (43)
- Rep → Replace substring (44)
- Char → String → char array (45)

## Key Focus
- String manipulation basics (reverse, palindrome, rotation)
- Counting operations (vowels, consonants, words, frequency)
- Duplicate handling (characters, words
