# Scenarios Module (Problems 107–121)

## Mnemonic
**“Merge Top Third Digit Match Sorted Rotate Long Longer Join CSV Distinct Salary Union Pairs.”**

- Merge → Merge arrays sorted (107)
- Top → Top 3 elements (108)
- Third → 3rd highest (109)
- Digit → Numbers starting with 1 (110)
- Match → allMatch/anyMatch/noneMatch (111)
- Sorted → Check list sorted (112)
- Rotate → Rotate list (113)
- Long → Longest string (114)
- Longer → Strings longer than n (115)
- Join → Comma separated string (116)
- CSV → Split CSV sorted (117)
- Distinct → Distinct preserving order (118)
- Salary → Salaries grouped sum (119)
- Union → Union & intersection (120)
- Pairs → All pairs summing to target (121)

- Real-world interview scenarios
- Combining streams, collectors, arrays, sets
