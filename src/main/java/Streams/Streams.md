# Streams Module (Problems 79–90)

## Mnemonic
**“Flat Flat Flat Skills Reduce Prefix Sum Peek Parallel Optional.”**

- Flat → Flatten nested list (79)
- Flat → FlatMap sentences (80)
- Flat → FlatMap list of lists (81)
- Skills → FlatMap employees skills (82)
- Reduce → Concatenate strings (83)
- Prefix → Longest common prefix (84)
- Sum → Reduce sum/max (85)
- Peek → Debugging pipeline (86)
- Parallel → Parallel streams (87)
- Optional → Avoid NPE, ifPresent, stream conversion (88–90)

## Key Focus
- flatMap for flattening nested structures
- reduce for aggregation (concat, prefix, sum, max)
- peek for debugging stream pipelines
- parallelStream for large data sets
- Optional to handle nulls safely
