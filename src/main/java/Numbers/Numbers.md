# Numbers Module (Problems 13–25)

## Mnemonic
**“Smart Reverse Prime Factor Fib Max Min Arm Pal Even Odd Avg Root.”**

- Smart → Sum of digits (13)
- Reverse → Reverse number (14)
- Prime → Prime check (15)
- Factor → Factorial (16)
- Fib → Fibonacci (17)
- Max Min → Largest/Smallest (18–19)
- Arm → Armstrong (20)
- Pal → Palindrome (21)
- Even Odd → Sum of even/odd (22–23)
- Avg → Average (24)
- Root → Square root (25)

## Key Focus
- Number theory basics (prime, factorial, Fibonacci)
- Special numbers (Armstrong, Palindrome)
- Stream operations on numeric lists
- Math utilities (average, sqrt, gcd)
