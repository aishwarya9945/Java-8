# Arrays Module (Problems 46–65)

## Mnemonic
**“Remove Sort First Skip Limit Rotate Partition Filter Clean Common Diff Unique Square Seq Rotate Zero Color Parity Remove.”**

- Remove → Remove duplicates (46)
- Sort → Asc/Desc (47)
- First → First element (48)
- Skip → Skip N (49)
- Limit → Limit N (50)
- Rotate → Rotate list (51)
- Partition → Even/Odd (52)
- Filter → Strings by length (53)
- Clean → Remove nulls/empty (54–55)
- Common → Common elements (56)
- Diff → Difference (57)
- Unique → Appearing once (58)
- Square → Squares (59)
- Seq → Longest consecutive sequence (60)
- Rotate → Array rotation (61)
- Zero → Move zeroes (62)
- Color → Sort colors (63)
- Parity → Sort by parity (64)
- Remove → Remove element (65)

## Key Focus
- Stream operations on lists (distinct, sorted, skip, limit, partitioningBy)
- Cleaning data (nulls, blanks)
- Set operations (common, difference, unique)
- Array algorithms (rotation, move zeroes, Dutch flag, parity sort)
- In-place modifications vs creating new arrays
