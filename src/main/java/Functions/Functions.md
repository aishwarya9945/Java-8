Predicate → Chain multiple Predicates (91)

Question: Demonstrate the difference between Function.andThen() and Function.compose() with numeric transformations.

Consumer → andThen chaining (93)  
Question: Chain multiple consumers (e.g., print a string and then convert it to uppercase).

Supplier → Lazy object creation (94)  
Question: Use a Supplier to create objects lazily (e.g., supply a new ArrayList).
