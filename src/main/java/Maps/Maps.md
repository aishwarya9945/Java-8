# Maps Module (Problems 66–78)

## Mnemonic
**“Word Char Most Top Freq Dup Filter Sort Compute Merge ToMap Mapping Collect.”**

- Word → Word frequency (66)
- Char → Character frequency (67)
- Most → Most frequent element (68)
- Top → Top K frequent (69)
- Freq → Word frequency again (70)
- Dup → Duplicate words (71)
- Filter → Filter map by value (72)
- Sort → Sort map by value descending (73)
- Compute → computeIfAbsent list map (74)
- Merge → merge for frequencies (75)
- ToMap → toMap with duplicate handling (76)
- Mapping → mapping inside groupingBy (77)
- Collect → collectingAndThen unmodifiable list (78)

## Key Focus
- Frequency counting (words, chars, duplicates)
- Filtering and sorting maps
- Advanced map operations (merge, computeIfAbsent)
- Collectors tricks (toMap, mapping, collectingAndThen)
